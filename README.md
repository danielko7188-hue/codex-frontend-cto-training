# Codex Frontend CTO Training

This repo packages the shareable parts of a frontend and UX/UI training system for Codex developers.

It is built around three things:

- a founder taste profile for what "premium" should feel like
- a reusable design-memory workspace with references, playbooks, scorecards, and templates
- curated study packs for product UI, AI UX, templates, and DESIGN.md-based style systems

## What is intentionally excluded

- local skill installation state from `~/.codex/`
- heavyweight raw repo mirrors under `design-memory/source-library/clones/`
- local copies of unrelated workspaces like `codex-main/` and `gstack/`

The public repo keeps the reusable training materials and curated notes. If you want to rehydrate optional source clones later, use the manifests and scripts already included under [design-memory/manifests](./design-memory/manifests) and [design-memory/tools](./design-memory/tools).
