# AI Product UX Pack: 2026-04-07

This pack is for AI-native product UX and UI.

Use it when the product includes:

- chat
- copilots
- AI workspaces
- agents
- tool calling
- approvals
- background AI jobs

## What this pack is for

- deciding whether AI should be chat, copilot, workflow, or agent
- making AI state and tool use visible
- designing approvals and human-in-the-loop flows
- turning AI output into useful artifacts instead of long blobs of text
- making AI products feel premium and trustworthy

## Recommended use order

1. Read the playbook first.
2. Pick the right interaction model.
3. Study 1 to 2 relevant interface sources, not all of them.
4. Review the result with the AI UX scorecard.
