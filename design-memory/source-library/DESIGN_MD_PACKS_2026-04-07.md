# DESIGN.md Packs: 2026-04-07

This pack is for structured design systems written in markdown for AI agents.

Use this when the question is not just "what components should I use?" but "how should I describe the visual system so an agent can build in the right style consistently?"

## Current pack

- [awesome-design-md](./design-md-packs/awesome-design-md.md)
- [minimalist-premium-founder-pack](./design-md-packs/minimalist-premium-founder-pack.md)

## What this teaches

- how strong products can be translated into DESIGN.md files
- how to describe visual systems in plain text for agent workflows
- how to reuse style direction without needing screenshots every time
- how to make design intent easier to carry across projects and threads

## Important limit

These packs are useful because they make design intent legible to AI agents.

They still need founder taste, product context, and judgment. A borrowed DESIGN.md should be adapted, not copied blindly.
