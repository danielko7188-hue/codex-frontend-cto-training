# Editorial Luxury

## Best For

- brand-forward sites
- founder brands
- storytelling-heavy product pages
- high-perception positioning

## Core Feel

Refined, composed, image-aware, high taste.

## Signals

- stronger contrast in typography scale
- more dramatic cropping and whitespace
- less repetitive component structure
- language that feels authored, not templated

## Avoid

- dashboard-like cards everywhere
- weak imagery
- timid type
