# Calm AI Workspace

## Best For

- copilots
- agent products
- AI-assisted drafting or review tools
- products that mix chat, artifacts, and approvals

## Core Feel

Calm, inspectable, trustworthy, tool-aware.

## Signals

- AI state is visible
- outputs are readable and editable
- tools and actions feel understandable
- interface supports real work, not just chat novelty

## Avoid

- mystical AI branding
- chat-only design when structure is needed
- flashy gradients standing in for product clarity
