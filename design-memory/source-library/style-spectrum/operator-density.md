# Operator Density

## Best For

- internal tools
- analyst and ops dashboards
- high-frequency workflows
- products where speed matters more than atmosphere

## Core Feel

Efficient, information-rich, sharp, controlled.

## Signals

- tighter spacing
- denser tables and panels
- stronger data hierarchy
- less decorative empty space

## Avoid

- giant card marketing UI
- oversized type scales
- consumer app emptiness
